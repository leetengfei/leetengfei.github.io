Bootstrap

简介

  • 容器 ： container (固定宽度) 或 container-fluid (100% 宽度)
  • 超大屏幕 : jumbotron (更大的标题，字体粗细 font-weight 被减为 200)
  • 行 : row 必须包含在 .container 或 .container-fluid 中，以便为其赋予合适的排列（aligment）和内补（padding）。
  • 列 : column 只有列可以作为行 row 的直接子元素，列内放内容

栅格参数

偏移 : offset

排列对齐

  • text-left : 向左对齐文本
  • text-center : 居中对齐文本
  • text-right : 向右对齐文本

图片

  • img-rounded : 圆角
  • img-circle : 圆形
  • img-thumbnail : 缩略图
  • img-responsive : 响应式

使用github和hexo搭建博客

使用github建立博客站点

1、注册github

2、新建仓库，仓库名为 username.github.io

　(username为你的用户名)

安装软件

1、Node.js

2、GitHub Desktop
(安装完成后登录github客户端)

配置SSH

打开 Git Shell，输入

ssh -T git@github.com

安装Hexo

输入 

npm install -g hexo-cli

安装完成后，输入

hexo init E:\hexo
cd E:\hexo
npm install

(E:\hexo目录可自己选择)

输入

hexo g
hexo s

此时可以访问网址 http://localhost:4000/ 本地预览

按 Ctrl + C 并输入 y 可以停止该服务

配置网站

安装Notepad++

使用Notepad++打开 _config.yml ，修改配置

type: git
repo: https://github.com/username/username.github.io.git

(冒号：后面有一个空格，username为你的用户名)

输入

npm install hexo-deployer-git --save

再输入

hexo clean
hexo g
hexo d

此时，可以通过https://username.github.io访问你的博客了

发表博文

输入

hexo n "文章题目"

此时 E:\hexo\source_posts中多出一个md文件，它就是新建的博文。

安装MarkdownPad2

使用MarkdownPad2编辑博文
保存后，输入

hexo clean
hexo g
hexo d

此时，博文就更新到了你的网站。

(以后更新博文需要先进入文件夹(cd E:\hexo)，再使用其他命令)

刚才所说的命令使用了简写，下面为简写与全称对照

hexo g == hexo generate
hexo d == hexo deploy
hexo s == hexo server
hexo n == hexo new

安装主题

从这里挑选喜欢的主题并根据说明文档安装

其他

可以下载安装此插件来使用本地的图片

你还可以继续优化，如添加评论框，浏览次数及绑定域名等

正则表达式

入门教程

正则表达式30分钟入门教程

某个字符串匹配某个正则表达式，通常是指这个字符串里有一部分（或几部分分别）能满足表达式给出的条件

特殊字符

如果需要这些字符用作文本字符，需要用反斜杠\对其进行转义。

分别为：(　)　[　]　{　}　\　^　$　.　|　*　+　?

模式 含义
() 标记一个子表达式的开始和结束位置
[] 表示一个匹配集合
{} 标记限定符表达式的开始和结束位置
\ 转义
. 匹配除了换行符(\n)以外的任意字符
| 逻辑或

常用

模式 含义
\d 匹配任何数字字符
\s 匹配任何空白字符
\w 匹配字母或数字或下划线或汉字

位置

模式 含义
\b 匹配一个单词边界
^ 匹配字符串的开始
$ 匹配字符串的结束
(?=exp) 匹配exp前面的位置
(?<=exp) 匹配exp后面的位置
(?!exp) 匹配后面不是exp的位置
(?<!exp) 匹配前面不是exp的位置

反义

模式 含义
\D 匹配任意非数字的字符
\S 匹配任意不是空白符的字符
\W 匹配任意不是字母，数字，下划线，汉字的字符
\B 匹配不是单词开头或结束的位置
[^x] 匹配除了x以外的任意字符

重复

模式 含义
* 重复零次或更多次
+ 重复一次或更多次
? 重复零次或一次
{n} 重复n次
{n,} 重复n次或更多次
{n,m} 重复n到m次

以上重复为贪婪模式，懒惰模式需要在任意一个重复限制符后加上一个?

即：*?　+?　??　{n,}?　{n,m}?

如何使用搜索引擎

基本

使用空格分割的关键词搜索

(一般来讲这样搜索更有效率，假如没找到，你也可以尝试用一句话来搜索)

重要

1、 使用谷歌搜索(梯子)

2、 使用英文搜索(假如具有英文阅读能力)

语法

1、 site: 在特定网站内搜索，其格式为：查询词+空格+site:网址

谷歌 搜索技巧 site:zhihu.com

2、 filetype: 特定文件搜索,其格式为：查询词+空格+filetype:格式

搜索技巧 filetype:doc

(常用的格式有:office(doc,docx,ppt,pptx,xls,xlsx)、Adobe(swf,pdf)等更多)

3、 ..: 数字范围，其格式为：查询词+空格+数字+..+数字

搜索技巧 2015..2016

(常用来限定时间)

4、 “”: 精确匹配，其格式为：“+查询词+”

"Google搜索技巧"

(返回包含“Google搜索技巧”内容的页面，不对查询词拆分)

5、 空格: 逻辑与，其格式为：查询词+空格+查询词

搜索 技巧

(返回同时包含两个关键词的页面)

6、 OR: 逻辑或，其格式为：查询词+空格+OR+空格+查询词

搜索 OR 技巧

(返回包含其中一个或两个关键词的页面)

7、 -: 逻辑非，其格式为：查询词+空格+-查询词(-号前加空格)

苹果 -日报

(返回包含苹果但不包含日报的页面)

8、 +: 强制搜索，其格式为：查询词+空格++查询词(+号前加空格)

分类

程序员

1、 直接复制提示的错误信息去搜索

2、 查询词加上所属的语言类别

排序　JavaScript

3、 站内搜索

查询词 site:stackoverflow.com

5、 学习某个东西，可以查询此词加下列查询词
