Bootstrap

简介

  • 容器 ： container (固定宽度) 或 container-fluid (100% 宽度)
  • 超大屏幕 : jumbotron (更大的标题，字体粗细 font-weight 被减为 200)
  • 行 : row 必须包含在 .container 或 .container-fluid 中，以便为其赋予合适的排列（aligment）和内补（padding）。
  • 列 : column 只有列可以作为行 row 的直接子元素，列内放内容

栅格参数

偏移 : offset

排列对齐

  • text-left : 向左对齐文本
  • text-center : 居中对齐文本
  • text-right : 向右对齐文本
  • text-justify : 对齐文本。该段落会根据屏幕的大小对超出屏幕的文字进行换行
  • text-nowrap : 该段落不会根据屏幕的大小对超出屏幕的文字进行换行

图片

  • img-rounded : 圆角
  • img-circle : 圆形
  • img-thumbnail : 缩略图
  • img-responsive : 响应式

图标

Glyphicon Halflings 的字体图标：

1、 在图标和文本之间添加一个空格

2、 不能在同一个元素上与其他类共同存在,应用到<span>标签上

3、 只能应用在不包含任何文本内容或子元素的元素上

比如

<span class="glyphicon glyphicon-search"></span>

情境文本颜色

  • text-muted : gray
  • text-primary : blue
  • text-success : green
  • text-info : light blue
  • text-warning : orange
  • text-danger : red
