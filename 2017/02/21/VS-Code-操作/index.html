VS Code 操作

界面

右下角可以迅速改变文件类型，新建文件后先保存

快捷键

  • Ctrl + Shift + P : 显示所有命令

编辑器

  • Ctrl + \ : 拆分编辑器

  • Ctrl + Shift + V : 预览效果

  • Ctrl + 1(2,3) : 选择当前激活的编辑器

  • Ctrl + Tab : 切换文件

  • Ctrl + W : 关闭编辑器

光标

  • Home : 移动到行首

  • End : 移动到行尾

  • Ctrl + Home : 移动到文件开头

  • Ctrl + End : 移动到文件结尾

  • Alt + click : 多行选择

其他

  • Ctrl + O : 打开文件

  • Ctrl + N : 新建文件

  • Ctrl + B : 显示隐藏侧边栏

  • Ctrl + F : 查找

  • CtrL + G : 跳转指定行

  • Shift + Alt + F : 格式化代码

HTML快捷输入

输入div按下Tab生成

<div></div>

输入<!按下Tab生成HTML5格式默认页面

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
       <title>Document</title>
</head>
<body>

</body>
</html>

支持Emmet插件

比如输入ul#nav>li.item$*4>a{item $}按下Tab生成

<ul id="nav">
    <li class="item1"><a href="">item 1</a></li>
    <li class="item2"><a href="">item 2</a></li>
    <li class="item3"><a href="">item 3</a></li>
    <li class="item4"><a href="">item 4</a></li>
</ul>
