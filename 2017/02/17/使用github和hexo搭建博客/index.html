使用github和hexo搭建博客

使用github建立博客站点

1、注册github

2、新建仓库，仓库名为 username.github.io

　(username为你的用户名)

安装软件

1、Node.js

2、GitHub Desktop
(安装完成后登录github客户端)

配置SSH

打开 Git Shell，输入

ssh -T git@github.com

安装Hexo

输入 

npm install -g hexo-cli

安装完成后，输入

hexo init E:\hexo
cd E:\hexo
npm install

(E:\hexo目录可自己选择)

输入

hexo g
hexo s

此时可以访问网址 http://localhost:4000/ 本地预览

按 Ctrl + C 并输入 y 可以停止该服务

配置网站

安装Notepad++

使用Notepad++打开 _config.yml ，修改配置

type: git
repo: https://github.com/username/username.github.io.git

(冒号：后面有一个空格，username为你的用户名)

输入

npm install hexo-deployer-git --save

再输入

hexo clean
hexo g
hexo d

此时，可以通过https://username.github.io访问你的博客了

发表博文

输入

hexo n "文章题目"

此时 E:\hexo\source_posts中多出一个md文件，它就是新建的博文。

安装MarkdownPad2

使用MarkdownPad2编辑博文
保存后，输入

hexo clean
hexo g
hexo d

此时，博文就更新到了你的网站。

(以后更新博文需要先进入文件夹(cd E:\hexo)，再使用其他命令)

刚才所说的命令使用了简写，下面为简写与全称对照

hexo g == hexo generate
hexo d == hexo deploy
hexo s == hexo server
hexo n == hexo new

安装主题

从这里挑选喜欢的主题并根据说明文档安装

其他

可以下载安装此插件来使用本地的图片

你还可以继续优化，如添加评论框，浏览次数及绑定域名等
